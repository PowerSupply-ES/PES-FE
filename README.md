# PES

url : https://pes23.com/

### 🚀 서비스 소개
- PowerSupply 동아리 내 신입생 SW교육 시스템
- 코딩테스트 채점 웹 프로그램
- 기존의 불편한 과제 채점 방식을 개선한 자체적인 서비스
- 일반적인 코딩테스트 서비스와 달리, 추가적인 질문을 통해 사용자의 이해도를 높임
- 신입생의 코드와 질문답변을 종합하여 재학생들은 pass/fail와 comment 제공
- 관리자모드를 통해 신입생 및 문제 정보들을 간편하게 수정


### 🖥️ 제공 기능
- 로그인, 회원가입
- 코딩문제 제공 및 채점
